# Config Files

The README contains the current user-level and project-level config layout.
This file is the stable home for a detailed reference.

## Current Entrypoints

- User config: `~/.gen/`
- Project config: `<project>/.gen/`
- Project memory: `GEN.md` or `CLAUDE.md`

Keep this reference in sync with `README.md` and `docs/reference/configuration.md`.
